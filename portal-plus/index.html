What is Portal Plus?

Portal Plus is a modernized Portal experience, built on the latest Source Engine SDK, utilizing the updated TF2 multiplayer branch. It features an updated UI, enhanced visuals, numerous bug fixes, and a plethora of brand-new features.

Our base, RGBSource, is a custom branch of the 2013 Source SDK, which has been heavily modified to support enhanced PBR materials, feature many quality-of-life improvements, and provide a more stable, enjoyable, and customizable experience.
